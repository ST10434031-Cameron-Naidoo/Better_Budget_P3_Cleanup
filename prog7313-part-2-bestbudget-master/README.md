# BetterBudget — BestBudget Team
### PROG7313 Part 2 | Android Budget Tracker App

---

##  App Overview
BetterBudget is a personal budget tracking Android app built in Kotlin. Users can create custom spending categories and subcategories, log expenses with receipt photos, and set monthly min/max goals to track their budget.

## Features
- Register and login with email and password
- Create, edit and delete spending categories with emoji icons
- Create subcategories under each category
- Log expenses with date, amount, description and receipt photo
- Set minimum and maximum monthly spending goals per category
- Filter and sort expenses and categories by date range
- View category breakdown with subcategory totals

---

## Architecture
- **Language:** Kotlin
- **Architecture:** MVVM (Model-View-ViewModel)
- **Database:** Room DB (SQLite)
- **Async:** Kotlin Coroutines
- **UI:** XML Layouts + Fragments
- **Min SDK:** 26

## How to Run
1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Run on an emulator (API 26+) or physical device
